    Connection Issue

    Toast Wallet is unable to connect either because your device has no network connectivity, or because the public gateway servers Toast Wallet uses to connect to the XRP Ledger are down or under heavy load—this has nothing to do with the XRP Ledger itself, only the servers that Toast Wallet uses to access the network.






    Add a custom gateway you would like Toast Wallet to connect to. A connection will be attempted and if it is successful the gateway will be stored and remembered as your preferred gateway.
